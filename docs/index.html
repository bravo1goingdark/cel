Engine

3.9KB. Zero deps.
Runs everywhere.

The core engine works in Node, Deno, Bun, and any browser. Import three functions and you're done.

npm cel-core v0.1.1 npm cel-export v0.1.1 npm cel-player v0.1.1
// install
$ npm install cel-core

// sample a scene at any timestamp
import { sampleScene } from 'cel-core';
const frame = sampleScene(scene, 1500);

// deterministic serialization
import { serialize } from 'cel-core';
const json = serialize(scene); // identical input = identical bytes

// validate with full error paths
import { validate } from 'cel-core';
const { valid, errors } = validate(input);
React

Drop-in component.
One import.

A ready-to-use component that plays any scene. Tree-shakeable, fully typed, 3KB.

npm cel-react v0.1.1 peer react ^18 || ^19
// install
$ npm install cel-react cel-core

// drop into any React app
import { CelAnimation } from 'cel-react';
import scene from './sleeping.aanim';

export default function App() {
  return <CelAnimation
    scene={scene}
    autoplay
    loop
    speed={1}
  />;
}
CelAnimation
High-level component with autoplay, loop, speed, width, height, and onFrame callback.
useCelPlayer
Low-level hook for custom render targets. Full player API: play, pause, seek, speed, destroy.
Format

.aanim files.
JSON you can read.

UTF-8 JSON with strict conventions. Deterministic output means every save produces an identical file. Perfect for version control.

{
  "$schema": "https://cel.dev/schema/v1.json",
  "version": 1,
  "duration": 4000,
  "fps": 60,
  "grid": { "cols": 60, "rows": 13 },
  "sprites": [{
    "id": "face",
    "text": "( -_- )",
    "keyframes": [
      { "t": 0, "x": 5.5, "y": 5, "easing": "inout" },
      { "t": 2000, "y": 5.6, "rotation": -1 },
      { "t": 4000, "y": 5, "rotation": 0 }
    ]
  }]
}

CLI

Validate, play, render.
From your terminal.

1.3MB static Rust binary. Batch export from YAML configs. Plays animations directly in the terminal with ANSI.

$ cel validate scenes/*.aanim
  ✓ sleeping.aanim
  ✓ spinner.aanim
  ✓ heartbeat.aanim
  ✓ 31 files validated, 0 errors

$ cel info presets/sleeping.aanim
  duration: 4000ms (4.0s)
  sprites:  4
  keyframes: 17 total

$ cel play presets/sleeping.aanim --loop
